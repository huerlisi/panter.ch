RECOMY ist eine online Plattform für Social Recruiting und Jobs. RECOMY verschafft Stelleninseraten gezielt Reichweite über soziale Netzwerke wie Facebook, XING, LinkedIn, Twitter und Google+ und platziert sie dort, wo sie Wirkung erzielen. So holt RECOMY potentielle Kandidaten gezielt dort ab, wo Sie sich täglich aufhalten: In den sozialen Netzwerken. Passive Stellensuchende, die gewöhnliche Stellenplattformen kaum besuchen, werden aktiviert.

Mit dem Externen-Referral-Programm von RECOMY kann sich jeder als externer Ambassador registrieren, automatisch vorselektierte Stelleninserate teilen und pro geteiltes Stelleninserat Geld verdienen. Dabei wird das reine Teilen honoriert, eine Zuteilungssoftware gewährleistet, dass die richtigen Ambassadoren die richtigen Stelleninserate weiterteilen.

Stellensuchenden liefert RECOMY Jobs, die auf das jeweilige Profil und Präferenzen passen

Kategorien & Begründung

Technology

  • OAuth Integration fünf sozialer Netzwerke (für Authentifizierung, den Import der Berufs- und Ausbildungserfahrung sowie um Stelleninserate zu teilen (letzteres für vier Netzwerke))
  • Single Page Application (basierend auf Backbone.js und Marionette)
  • RESTful APIs im Hintergrund
  • Komplexe PayPal Integration für Ein- und Auszahlungen
  • Integration diverser Recruiting-Lösungen, um den Zugang für die Kunden zu vereinfachen
  • Automatischer Import von Stelleninseraten diverser Unternehmen via Web-interface
  • Integration von Nominatim (auch verwendet in OpenStreetMaps und MapQuest) für Job-Geolocation
  • Integration von bookyt für alle buchalterischen Belange
  • Solr für Suchfunktion und Job-Matching (Solr wird verwendet von http://wiki.apache.org/solr/PublicServers)
  • SEO-implementiert via Prerender Gem, damit die Single Page-Applikation von Suchmaschinen-Bots gefunden wird
  • Strukturierte, agile Entwicklung: PivotalTracker für Projektmanagement, Github für CVS, Github Pull Requests für die Code Reviews, New Relic für das Applikations-Monitoring, Rollbar für das Fehler-Tracking, TravisCI Pro für die kontinuierliche Integration, CodeClimate für die Code-Qualitätsprüfung
  • Umfassende Test-Suite: 1500 Jasmine Specs und 2000 rspec Specs für Frontend- und Backend-Testing der Anwendung.

Innovation

  • Edge-Rank-bereinigtes Targeting von Personen über soziale Netzwerke.
  • Visionärer Ansatz in Bezug auf die Nutzung sozialer Beziehungsnetzwerke: Qualifizierte Ambassadoren-Profile als Schnittstelle zur gezielten Weiterverbreitung von z.B. Stelleninseraten.
  • Frontend als Singlepage Application: massiv verkürzte Ladezeiten, schnelleres Navigieren.

Usability

  1. Drei Haupt-Zielgruppen stehen im Fokus, begleitet von einer sekundären Vierten

    1. Aktive Stellensuchende, die von massgeschneiderten Jobangeboten gefunden werden möchten oder eine Alternative suchen zu klassischen online Stellenplattformen.
    2. Unternehmen auf Kandidatensuche / Unternehmen mit offenen Vakanzen, die für wenig Budget maximalen Erfolg erzielen möchten und es Leid sind, Stelleninserate auf online Stellenplattformen zu schalten, wo sie von Usern gefunden werden müssen, stattdessen aber eine Alternative suchen, Stelleninserate proaktiv pushen zu können, sodass diese zu den richtigen Kandidaten hinausgetragen werden.
    3. Ambassadoren: Aktive User von Social Media, Generation Y, die ohne viel Aufwand Geld verdienen wollen oder einen Weg suchen, wie sich das eigene Beziehungsnetzwerk auch finanziell auszahlt, ohne dass man die Daten an Dritte verkauft.
    4. Passive / latent Stellensuchende auf sozialen Netzwerken (werden nicht auf der Website angesprochen).

  2. Benutzerfreundlichkeits-Merkmale der Website

    • Intuitiver und zielgruppen-segmentierter Aufbau der Website (Segmentation, Assoziative Bildsprache, natürlicher Blickverlauf im Seitenaufbau berücksichtigt, psycho-somatisch günstig platzierte und eingefärbte Call-to-action-Elemente / Buttons, klare visuelle Trennung von aktiven, passiven oder bereits benutzten Seitenelementen.
    • Geringe Anzahl Klicks nötig, um zum Ziel zu kommen.
    • Die Möglicheit, als Kandidat einfach und schnell ein Bewerbungs-Video aufzunehmen und online zu stellen.

Business

Generell

Hoher ROI, geringe Investitionskosten, wenig Zeitaufwand nötig seitens der Recruiter: Erstens bieten die Referral-Programme Recruitern dank der “Push” Funktion globale Reichweite für ein Stelleninserat über Facebook, Xing, LinkedIn, Twitter, Google+.

Ebenso werden passive Stellenkandidaten erreicht, da die Stelleninserate als Statusmitteilung erscheinen. Zweitens erlauben Echtzeit-Statistiken einen Einblick in die Performance eines Inserates (Views, Klicks und Bewerbungen pro Stelleninserat). Das vom Recruiter eingesetzte Budget definiert die Reichweite des Inserates, im Sinne von „pay for performance only“

Zukunftschancen

Das Angebot besitzt eine hohe Attraktivität, aufgrund seiner zukunftsträchtigen Konzeption, vor allem vor dem Hintergrund der heutigen Wichtigkeit sozialer Netzwerke sowie des generellen Social Recruiting Hypes.

Grundsätzlich tendiert Social Recruiting momentan zu proaktiven Rekrutierungslösungen, so besteht denn mit RECOMY auch keine Abhängigkeit von Plattformbesuchern, so wie bei herkömmlichen Stellenplattformen, da das Stelleninserat mittels “Push”-Programmen (Internal- und External-Referral-Program) über die Beziehungsnetzwerke hinausgetragen wird.

Aufgrund dieser Tatsachen wird eine hohe Nutzerakzeptanz erwartet, da RECOMY einfach handhabbar ist, auf bestehenden Ressourcen aufbaut und die wesentlichen Prozesse im Hintergrund ablaufen und sich das Potential sozialer Beziehungsnetzwerke für die Kandidatensuche auf einer Website konsolidieren lässt.
